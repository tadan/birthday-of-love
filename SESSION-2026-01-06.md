### 3. Form Features Implemented

**Fields:**

- Your address (required when attending)

- When "Yes, can't wait!" selected: All fields visible

## ⚠️ Known Issues & Solutions

### Issue 1: Radix Dialog Wouldn't Display
**Problem**: Complex Radix UI Dialog component had z-index and visibility issues
**Solution**: Created simple custom modal with fixed positioning and conditional rendering

### Issue 2: CSS Not Applying to Inputs
**Problem**: Tailwind classes not overriding default input styles
**Solution**: Used inline styles with explicit `border: '1px solid lightgray'`

### Issue 3: Radio Buttons Not Triggering Conditional Logic
**Problem**: react-hook-form `watch` not triggering re-renders
**Solution**: Used local state (`attending`) with `onChange` handlers on native radio inputs

### Issue 4: Initial Formspree Endpoint 404
**Problem**: Endpoint `xjgknv` didn't exist
**Solution**: Created new form in Formspree, updated to `xjgknvop`
